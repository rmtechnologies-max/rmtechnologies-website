Enterprise FMCG Data Platform

Flagship Project

Enterprise Vendor Data Platform

CLIENT: Leading FMCG Manufacturer · INDUSTRY: Food & Beverages

Designed and deployed an end-to-end enterprise data platform for one of India's fastest-growing FMCG companies. The platform centralized order management, vendor analytics, and sales reporting across 5000+ active vendors processing millions of orders monthly.

Challenge

Data from 5000+ vendors flowed through multiple channels — vendor databases, REST APIs, purchase systems, websites, and even SMS-based ordering. No centralized analytics, no real-time visibility, and manual reporting taking days.

Solution

Built a Medallion Architecture (Bronze → Silver → Gold) on AWS Databricks with Kafka and REST API ingestion, PySpark transformations, Apache Airflow orchestration, Delta Lake optimization, automated data quality checks, and CI/CD deployment pipelines.
